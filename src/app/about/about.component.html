About Mutlify...

Multifly is a simple flash-card style math game for children (or anyone) to learn multiplication. It has a few game mechanics built into it such as level difficulty, progression and gem currency.

Why?

Because there isn't a site out there that doesn't have ads or require users to sign up. This is intended to be used by young children.

Levels

Each level gets progressivly harder. Each level requires more and more correct answers to level up.

Streaks

Streaks are when you get 5 or more answers correct in a row!

Gems

You earn 5 gems when you level up. You also earn 3 gems when you achieve a streak!.

Store

You will be able to eventually buy stuff with your gems. Maybe customize a character or buy charms or something. TBD!
